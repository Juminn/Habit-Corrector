## environment:
- python 3.7
- tensorflow-gpu 1.14.0
- tensorflow-probability 0.7.0
- dm-sonnet 1.34
- CUDA 10.0
- CuDNN 7.4

1. Detect habits through cam
2. Helps users stop a habit by notifying them
